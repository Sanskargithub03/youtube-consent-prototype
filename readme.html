RECPRODUCT THINKING CASE STUDY · VEDANTU PM INTERNSHIP

Designing consent for someone
who isn't old enough to give it.

A redesign of YouTube India's sign-up and consent flow for minors, built to satisfy the Digital Personal Data Protection (DPDP) Act, 2023 — without wrecking the product for everyone else.

Sanskar · ECE, IIIT Nagpur Focus: minors' consent flow Region: India
00:01 — THE PROBLEM

Age is self-declared. Nothing checks it.

YouTube's sign-up asks for a birth date and takes it on faith. A 14-year-old can type in an adult year and get an account whose watch history, search history and location start feeding recommendations and ad-targeting — with no parent ever in the loop, and no consent that would hold up under DPDP's requirement for verifiable parental consent.

₹250 crore — the maximum penalty under DPDP for mishandling a child's personal data. This isn't a UX nice-to-have, it's a legal exposure.
00:04 — THE SOLUTION

Privacy-by-default, not perfect — proportionate.

Design principle: default to the safest state, and nudge for correction instead of demanding permission upfront.

04:01

Minimal, on-device signal

A lightweight signal flags likely-minor accounts — computed on-device where possible, used only to route to a supervised experience, never reused for ads. Disclosed in the privacy notice, not silent.

04:02

Toggles start off

Recommendations, ad personalisation, third-party sharing — every optional setting starts OFF for a flagged account. A parent turns on only what they actively want.

04:03

Dashboard + 6-month nudge

Consent controls live inside YouTube itself, not a separate app — with a proactive check-in every six months, not just once at sign-up.

04:04

Soft self-correction

For accounts already mis-declared as adult, a periodic, non-punitive prompt routes to consent only if confirmed — good faith over mass re-verification.

What this doesn't solve: a determined teen can still lie through the soft prompt, and signal-based detection isn't foolproof. This trades certainty for proportionate, privacy-respecting friction.

00:07 — WALK THE FLOW

Try it yourself

The happy path, end to end: sign-up → flagged as a minor → parent consent → confirmation → parent dashboard → data rights → periodic re-consent.
